##################################################2. Typography ##############################################
1. Headings and Typography:

h1, h2, h3, h4, h5, h6: Default HTML headings for different text sizes.

.h1 to .h6: Apply heading styles to any element (e.g., <p>).(Bootstrap এর class use করে Bold হবে।)

.display-1 to .display-6: Larger, prominent headings for hero sections.(নরমাল Bold এর চেয়ে বেশী Bold হবে।)


2. Text Utilities:

.lead: Styles a paragraph with increased font size and lighter weight for emphasis.

.blockquote: Styles a blockquote with indentation for quotes.



3. Font Weight & Style:

.fw-bold to .fw-lighter: Adjusts text weight from bold to light.
.fst-italic: Applies italic style.
.fst-normal: Resets to normal font style.



4. Line Height:

.lh-1,.lh-2 .lh-sm, .lh-base, .lh-lg: Controls line height (1x, small, base, large).



5. Text Transform:

.text-lowercase: Converts text to lowercase.
.text-uppercase: Converts text to uppercase.
.text-capitalize: Capitalizes the first letter of each word.


6. Text Alignment:(container এ text কোথায় বসবে তা সেট করা যায়।)

.text-start, .text-center, .text-end: Aligns text to the left, center, or right.


text-sm-end==>screen size 576px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
text-md-end==>screen size 768px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
text-lg-end==>screen size 992px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
text-xl-end==>screen size 1200px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
Responsive Alignment: .text-sm-end, .text-md-end, .text-lg-end, .text-xl-end align text based on screen size breakpoints.

################################################## 3. Display & Position ##############################################
<div class="position-static">...</div>
Default positioning (no special positioning applied).
normally যেখানে বসার কথা সেখানে বসবে।

<div class="position-relative">...</div>
Positioned relative to its normal position. Child elements with absolute positioning use this as their reference.
(যেখানে বসার কথা সেখানে বসবে তবে child element বসবে relative কে ফলো করে।)

<div class="position-absolute">...</div>
Positioned relative to the nearest positioned ancestor (relative, absolute, or fixed).
(এক্ষেত্রে relative parent element এর child হিসেবে absolute ব্যবহার করতে হয়)

<div class="position-fixed">...</div>
Positioned relative to the viewport and remains in place during scrolling.
(ফিক্সড পজিশন হিসেবে element বসবে। যেখানে element বসাব সেখানে বসবে।)

		<!-- Relative and Absolute Positioning -->
        <div class="position-relative bg-primary text-white p-4 mb-4" style="height: 150px;">

            <div class="position-absolute top-0 start-0 bg-danger text-white p-2">
                I'm positioned absolutely within the relative container.
            </div>
        </div>

<div class="position-sticky">...</div>
(Sticks to a defined offset (top, bottom, etc.) as you scroll but only within its parent container.)
(page scroll করলে যেখানে সেট করব সেখানে ফিক্সড হবে।)
(তবে parent element এর viewport full হতে হবে।)

 <!-- Parent -->
<div class="container-md">
    <!-- Sticky Positioning -->
    <div class="position-sticky top-0 bg-warning text-dark p-3" style="z-index: 1020;">
        <p class="mb-0">I stick to the top when scrolling!</p>
    </div>
</div>



(element কোথায় সেট করব তার পজিশন ফিক্সড করা হয় top,start,bottom,end এর মাধ্যমে।)
top - for the vertical top position
start - for the horizontal left position (in LTR)
bottom - for the vertical bottom position
end - for the horizontal right position (in LTR)
Where position is one of:

0 - for 0 edge position
50 - for 50% edge position
100 - for 100% edge position

		top-0 end-0(top-right corner element set হবে।)
		top-0 start-0(top-left corner element set হবে।)
		bottom-0 start-0(bottom-left corner element set হবে।)
		bottom-0 end-0(bottom-right corner element set হবে।)
		
		<!-- Relative and Absolute Positioning -->
        <div class="position-relative bg-primary text-white p-4 mb-4" style="height: 150px;">

            <div class="position-absolute top-0 end-0 bg-danger text-white p-2">
                I'm positioned absolutely within the relative container.
            </div>
        </div>




Center elements:
In addition, you can also center the elements with the transform utility class .translate-middle.
This class applies the transformations translateX(-50%) and translateY(-50%) to the element which, in combination with the edge positioning utilities, allows you to absolute center an element.
(translate-middle :
	top-0 start-0 translate-middle এর মাধ্যমে element বসবে top-left corner এ তবে element centered হবে কারণ এখানে left এ ৫০% ‍যাবে এবং উপরে এর দিকে ৫০% ‍যাবে।
	top-100 start-100 translate-middle এর মাধ্যমে element বসবে bottom-left corner এ তবে element centered হবে কারণ এখানে left এ ৫০% ‍যাবে এবং উপরে এর দিকে ৫০% ‍যাবে।
	
	translate-middle:
		Applies CSS transformations:
		translateX(-50%): Moves the element 50% of its width to the left.
		translateY(-50%): Moves the element 50% of its height upwards.


	<div class="position-relative bg-light mt-5 bg-secondary" style="height: 300px;">
        <div class="position-absolute top-0 start-0 translate-middle p-3 bg-primary">XYZ</div>
		<div class="position-absolute top-100 start-100 translate-middle p-3 bg-primary">XYZ</div>
    </div>




By adding .translate-middle-x or .translate-middle-y classes, elements can be positioned only in horizontal or vertical direction.
(horizontal এবং vertically center করার জন্য element ব্যবহৃত হবে।)
	
	 <div class="position-relative bg-light mt-5" style="height: 300px;">
       
        <div class="position-absolute top-0 start-50 translate-middle-x  bg-danger p-3">XYZ</div>
        
        <div class="position-absolute top-50 start-50 translate-middle bg-danger p-3">XYZ</div>
        
        <div class="position-absolute bottom-0 start-50 translate-middle-x bg-danger p-3">XYZ</div>
        
     </div>

d-none: Hides the element, making it invisible and removing it from the document flow.
(d-none দিলে element টি hidden থাকবে।)

d-inline: Makes the element behave like an inline element, allowing other elements to sit on the same line.
(d-inline দিলে block element হয়ে যাবে inline)

d-block: Converts the element to a block-level element, making it occupy the full width of its container.
(inline element হয়ে যাবে block element)

d-inline-block: Combines the characteristics of inline and block elements. The element remains inline but allows setting height and width.
(d-inline-block এর মাধ্যমে element inline হবে তবে এক্ষেত্রে inline element এর size দিতে পারব।)


(screen size এর উপর ভিত্তি করে block/none করা যাবে।)
md: 720px
screen size 768px এর উপরে গেলে display block/display-none বসবে।

d-md-block: Displays the element as a block starting from the md (medium) breakpoint and above.

d-md-none: Hides the element starting from the md (medium) breakpoint and above.

#################################################################5. Spacing######################################################

Where property is one of:( margin ও padding )

m - for set margin
p - for set padding

Where sides is one of:

t - for set margin-top or padding-top(top থেকে margin/padding হবে )
b - for set margin-bottom or padding-bottom(bottom থেকে margin/padding হবে)
s - (start) for  set margin-left or padding-left (left থেকে margin/padding হবে)
e - (end) for  set margin-right or padding-right (end থেকে margin/padding হবে)
x - for classes that set both *-left and *-right (left ও right উভয় থেকে margin/padding হবে)
y - for classes that set both *-top and *-bottom (top ও bottom উভয় থেকে margin/padding হবে)
blank - for classes that set a margin or padding on all 4 sides of the element (top থেকে margin/padding হবে)
Where size is one of:

0 - 0
1 -  .25rem
2 -  .5rem
3 -  .1rem
4 -  1.5rem
5 -    3rem
auto - for classes that set the margin to auto


margin,padding apply:
m-1/p-1 (Margin/Padding All(সব দিক(top,end,bottom,start) থেকে margin apply হবে।) 
my-1/py-1 (Margin/Padding Top & Bottom (my/py দিয়ে margin/padding => Top Bottom এ apply হবে।)
mx-1/px-1(Margin/Padding Left & Right (mx দিয়ে margin => left right এ apply হবে।))
mt-1/pt-1(Margin/Padding Top (mt/pt দিয়ে margin/padding => top এ apply হবে।))
mb-1/pb-1 (Margin/Padding Bottom (mb/pb দিয়ে margin/padding => bottom এ apply হবে।)
ms-1/ps-1(Margin/Padding Left/Start)
me-1/pe-1 (Margin/Padding right/end)

(Block/Flex Box এ margin-auto use হয়।)
m-auto: Automatically centers the element in both vertical and horizontal directions.
(Block/Flex Box এর মধ্যে content কে horizontally ও vertically center এ নিয়ে আসা যায়।)
	<div class="text-bg-warning w-50 m-auto">Lorem, ipsum dolor.</div>
    <div class="d-flex mb-5" style="height: 200px;border: 1px solid red">
        <div class="text-bg-warning w-50 m-auto">Lorem, ipsum dolor.</div>
    </div>
my-auto: Vertically centers the element within its parent.
(Block/Flex Box এর মধ্যে content কে  vertically center এ নিয়ে আসা যায়।)

	<div class="text-bg-primary w-50 my-auto">Lorem, ipsum dolor.</div>
    <div class="d-flex mb-5" style="height: 200px;border: 1px solid red">
        <div class="text-bg-primary w-50 my-auto">Lorem, ipsum dolor.</div>
    </div>	

mx-auto: Horizontally centers the element within its parent.
(Block/Flex Box এর মধ্যে content কে  horizontally center এ নিয়ে আসা যায়।)
	<div class="text-bg-info w-50 mx-auto">Lorem, ipsum dolor.</div>
    <div class="d-flex mb-5" style="height: 200px;border: 1px solid red">
        <div class="text-bg-success w-50 mx-auto">Lorem, ipsum dolor.</div>
    </div>
	

ms-auto: Pushes the element to the right (in LTR) by setting an automatic margin on the left side.
(Block/Flex Box এর মধ্যে content কে  right এ নিয়ে যাবে।(Right Align)))
	<div class="text-bg-danger w-50 ms-auto">Lorem, ipsum dolor.</div>
    <div class="d-flex mb-5" style="height: 200px;border: 1px solid red">
        <div class="text-bg-danger w-50 ms-auto">Lorem, ipsum dolor.</div>
    </div>


me-auto: Pushes the element to the left (in LTR) by setting an automatic margin on the right side.
(Block/Flex Box এর মধ্যে content কে  left এ নিয়ে যাবে।(Left Align)))

	<div class="text-bg-dark w-50 me-auto">Lorem, ipsum dolor.</div>
    <div class="d-flex mb-5" style="height: 200px;border: 1px solid red">
        <div class="text-bg-dark w-50 me-auto">Lorem, ipsum dolor.</div>
    </div>

#################################################################06-sizing-borders-shadows######################################################
Border Classes:
border: Adds a border to all sides.
	<div class="p-3 mb-2 bg-light border">Regular</div>
	
border-{color}: Sets the border color (e.g., border-primary, border-secondary, border-success, etc.).
	<div class="p-3 mb-2 bg-light border border-primary">Primary</div>
    <div class="p-3 mb-2 bg-light border border-primary-subtle">Primary Subtle</div>
	
border-{size}: Sets the border thickness (e.g., border-1, border-2, border-3, border-5).
	<div class="border border-1">Border size 1</div>
    <div class="border border-2">Border size 2</div>
	
border-{side}: Adds a border to a specific side (e.g., border-top, border-end, border-bottom, border-start).
	<div class="p-3 mb-2 bg-light border-top">Border Top</div>
    <div class="p-3 mb-2 bg-light border-bottom">Border Bottom</div>
    <div class="p-3 mb-2 bg-light border-end">Border Right</div>
    <div class="p-3 mb-2 bg-light border-start">Border Left</div>
border-{side}-0: Removes the border from a specific side (e.g., border-top-0, border-end-0).
	<div class="border  bg-light  border-0 mb-1">No border</div>
    <div class="border bg-light  border-top-0 mb-1">No top border</div>
    <div class="border  bg-light border-end-0 mb-1">No right border</div>
    <div class="border bg-light  border-bottom-0 mb-1">No bottom border</div>
    <div class="border  bg-light border-start-0 mb-1">No left border</div>


Rounded Border Classes:
rounded: Adds default rounded corners to all sides.
	<div class="p-3 mb-2 bg-light border rounded">Rounded Default</div>
	
rounded-{size}: Adjusts the roundness of the corners (e.g., rounded-0, rounded-1, rounded-2, rounded-3, rounded-4, rounded-5).
	<div class="p-3 mb-2 bg-light border rounded-0">Rounded 0</div>
    <div class="p-3 mb-2 bg-light border rounded-1">Rounded 1</div>
    <div class="p-3 mb-2 bg-light border rounded-2">Rounded 2</div>
	
rounded-{side}: Rounds specific sides (e.g., rounded-top, rounded-end, rounded-bottom, rounded-start).
	<div class="p-3 mb-2 bg-light border rounded">Rounded Default</div>
    <div class="p-3 mb-2 bg-light border rounded-top">Rounded Top</div>
    <div class="p-3 mb-2 bg-light border rounded-bottom">Rounded Bottom</div>
    <div class="p-3 mb-2 bg-light border rounded-end">Rounded Right</div>
    <div class="p-3 mb-2 bg-light border rounded-start">Rounded Left</div>
	
rounded-circle: Creates a circular border.
	<div class="p-3 mb-2 bg-light w-25 border rounded-circle">Rounded Circle</div>
rounded-pill: Creates a pill-shaped border.
	<div class="p-3 mb-2 bg-light w-25 border rounded-pill">Rounded Circle</div>

Special Combined Classes:
	<div class="w-25 p-5 rounded-bottom-1 bg-primary mb-3">75*75</div>
    <div class="w-25 p-5 rounded-start-2 bg-danger mb-3">75*75</div>
    <div class="w-25 p-5 rounded-end-circle bg-info mb-3">75*75</div>
    <div class="w-25 p-5 rounded-start-circle bg-info mb-3">75*75</div>
    <div class="w-25 p-5 rounded-start-pill bg-success mb-3">75*75</div>
    <div class="w-25 p-5 rounded-top-0 rounded-5 bg-success mb-3">75*75</div>

Shadow:
    <div class="p-3 mb-5 shadow-none">No shadow</div>
    <div class="p-3 mb-5 shadow-sm">Small shadow</div>
    <div class="p-3 mb-5 shadow">Regular shadow</div>
    <div class="p-3 mb-5 shadow-lg">Larger shadow</div>
