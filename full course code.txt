##################################################2. Typography ##############################################
1. Headings and Typography:

h1, h2, h3, h4, h5, h6: Default HTML headings for different text sizes.

.h1 to .h6: Apply heading styles to any element (e.g., <p>).(Bootstrap এর class use করে Bold হবে।)

.display-1 to .display-6: Larger, prominent headings for hero sections.(নরমাল Bold এর চেয়ে বেশী Bold হবে।)


2. Text Utilities:

.lead: Styles a paragraph with increased font size and lighter weight for emphasis.

.blockquote: Styles a blockquote with indentation for quotes.



3. Font Weight & Style:

.fw-bold to .fw-lighter: Adjusts text weight from bold to light.
.fst-italic: Applies italic style.
.fst-normal: Resets to normal font style.



4. Line Height:

.lh-1,.lh-2 .lh-sm, .lh-base, .lh-lg: Controls line height (1x, small, base, large).



5. Text Transform:

.text-lowercase: Converts text to lowercase.
.text-uppercase: Converts text to uppercase.
.text-capitalize: Capitalizes the first letter of each word.


6. Text Alignment:(container এ text কোথায় বসবে তা সেট করা যায়।)

.text-start, .text-center, .text-end: Aligns text to the left, center, or right.


text-sm-end==>screen size 576px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
text-md-end==>screen size 768px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
text-lg-end==>screen size 992px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।
text-xl-end==>screen size 1200px এর উপরে গেলে আউটপুট এ content/text গুলো শেষে বসবে।

################################################## 3. Display & Position ##############################################
<div class="position-static">...</div>
Default positioning (no special positioning applied).
normally যেখানে বসার কথা সেখানে বসবে।

<div class="position-relative">...</div>
Positioned relative to its normal position. Child elements with absolute positioning use this as their reference.
(যেখানে বসার কথা সেখানে বসবে তবে child element বসবে relative কে ফলো করে।)

<div class="position-absolute">...</div>
Positioned relative to the nearest positioned ancestor (relative, absolute, or fixed).
(এক্ষেত্রে relative parent element এর child হিসেবে absolute ব্যবহার করতে হয়)

<div class="position-fixed">...</div>
Positioned relative to the viewport and remains in place during scrolling.
(ফিক্সড পজিশন হিসেবে element বসবে। যেখানে element বসাব সেখানে বসবে।)

		<!-- Relative and Absolute Positioning -->
        <div class="position-relative bg-primary text-white p-4 mb-4" style="height: 150px;">

            <div class="position-absolute top-0 start-0 bg-danger text-white p-2">
                I'm positioned absolutely within the relative container.
            </div>
        </div>

<div class="position-sticky">...</div>
(Sticks to a defined offset (top, bottom, etc.) as you scroll but only within its parent container.)
(page scroll করলে যেখানে সেট করব সেখানে ফিক্সড হবে।)
(তবে parent element এর viewport full হতে হবে।)

 <!-- Parent -->
<div class="container-md">
    <!-- Sticky Positioning -->
    <div class="position-sticky top-0 bg-warning text-dark p-3" style="z-index: 1020;">
        <p class="mb-0">I stick to the top when scrolling!</p>
    </div>
</div>



(element কোথায় সেট করব তার পজিশন ফিক্সড করা হয় top,start,bottom,end এর মাধ্যমে।)
top - for the vertical top position
start - for the horizontal left position (in LTR)
bottom - for the vertical bottom position
end - for the horizontal right position (in LTR)
Where position is one of:

0 - for 0 edge position
50 - for 50% edge position
100 - for 100% edge position

		top-0 end-0(top-right corner element set হবে।)
		top-0 start-0(top-left corner element set হবে।)
		bottom-0 start-0(bottom-left corner element set হবে।)
		bottom-0 end-0(bottom-right corner element set হবে।)
		
		<!-- Relative and Absolute Positioning -->
        <div class="position-relative bg-primary text-white p-4 mb-4" style="height: 150px;">

            <div class="position-absolute top-0 end-0 bg-danger text-white p-2">
                I'm positioned absolutely within the relative container.
            </div>
        </div>




Center elements:
In addition, you can also center the elements with the transform utility class .translate-middle.
This class applies the transformations translateX(-50%) and translateY(-50%) to the element which, in combination with the edge positioning utilities, allows you to absolute center an element.
(translate-middle :
	top-0 start-0 translate-middle এর মাধ্যমে element বসবে top-left corner এ তবে element centered হবে কারণ এখানে left এ ৫০% ‍যাবে এবং উপরে এর দিকে ৫০% ‍যাবে।
	top-100 start-100 translate-middle এর মাধ্যমে element বসবে bottom-left corner এ তবে element centered হবে কারণ এখানে left এ ৫০% ‍যাবে এবং উপরে এর দিকে ৫০% ‍যাবে।
	
	translate-middle:
		Applies CSS transformations:
		translateX(-50%): Moves the element 50% of its width to the left.
		translateY(-50%): Moves the element 50% of its height upwards.


	<div class="position-relative bg-light mt-5 bg-secondary" style="height: 300px;">
        <div class="position-absolute top-0 start-0 translate-middle p-3 bg-primary">XYZ</div>
		<div class="position-absolute top-100 start-100 translate-middle p-3 bg-primary">XYZ</div>
    </div>




By adding .translate-middle-x or .translate-middle-y classes, elements can be positioned only in horizontal or vertical direction.
(horizontal এবং vertically center করার জন্য element ব্যবহৃত হবে।)
	
	 <div class="position-relative bg-light mt-5" style="height: 300px;">
       
        <div class="position-absolute top-0 start-50 translate-middle-x  bg-danger p-3">XYZ</div>
        
        <div class="position-absolute top-50 start-50 translate-middle bg-danger p-3">XYZ</div>
        
        <div class="position-absolute bottom-0 start-50 translate-middle-x bg-danger p-3">XYZ</div>
        
     </div>

d-none: Hides the element, making it invisible and removing it from the document flow.
(d-none দিলে element টি hidden থাকবে।)

d-inline: Makes the element behave like an inline element, allowing other elements to sit on the same line.
(d-inline দিলে block element হয়ে যাবে inline)

d-block: Converts the element to a block-level element, making it occupy the full width of its container.
(inline element হয়ে যাবে block element)

d-inline-block: Combines the characteristics of inline and block elements. The element remains inline but allows setting height and width.
(d-inline-block এর মাধ্যমে element inline হবে তবে এক্ষেত্রে inline element এর size দিতে পারব।)


(screen size এর উপর ভিত্তি করে block/none করা যাবে।)
md: 720px
screen size 768px এর উপরে গেলে display block/display-none বসবে।

d-md-block: Displays the element as a block starting from the md (medium) breakpoint and above.

d-md-none: Hides the element starting from the md (medium) breakpoint and above.
