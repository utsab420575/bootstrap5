-------------------------------------------------(4.1) 1. Dropdowns------------------------------------------------------------


.dropdown: Wrapper class for creating a dropdown component.

.dropdown-toggle: Class for the button that toggles the dropdown menu.

.dropdown-menu: Class for the container of dropdown items.

.dropdown-item: Class for individual items inside the dropdown menu.

.btn-group: Class for grouping buttons, used in split dropdowns.

.dropdown-toggle-split: Class for the split button's dropdown arrow.

.dropdown-menu-dark: Class for applying a dark theme to the dropdown menu.

.dropup: Class for making the dropdown open upwards instead of downwards.

.visually-hidden: Class for hiding text visually while keeping it accessible to screen readers.

.btn: Base class for styling buttons (e.g., primary, danger, dark).

Single Button Dropdown: A basic dropdown with a single button that toggles the dropdown menu.

Split Button Dropdown: A dropdown with a split button, where one part is the primary action and the other part toggles the dropdown.

Dark Theme Dropdown: A dropdown with a dark-themed menu.

Dropup: A dropdown that opens upwards instead of downwards.

Bootstrap JavaScript bundle: The script that enables Bootstrap's interactive components, including dropdowns.

-------------------------------------------(4.2)  2. Accordions & Collapse-----------------------------------------------------
Main Accordion Container:(এটাই parent container , এর মধ্যে মূলত সব ডাটা থাকবে।)

	The .accordion class is the wrapper for the entire accordion component.

	The id="accordionExample" is used to group the accordion items together.

Accordion Item:(এটা দিয়ে individual item গুলো নিয়ে কাজ করা হয়।)

	Each .accordion-item represents a single section of the accordion.

Accordion Header:(individual item এর header যেখানে button থাকে )

	The .accordion-header contains the toggle button (.accordion-button) for expanding/collapsing the content.

Toggle Button:(এর মাধ্যমে collapse/expand enable হয় এবং data-bs-target দ্বারা কি document show হবে তা select করা যায়।)

	The .accordion-button is the clickable button that toggles the visibility of the associated .accordion-collapse element.

	The data-bs-toggle="collapse" enables the collapse/expand functionality.

	The data-bs-target links the button to the specific collapsible element (e.g., #collapseOne).

Collapsible Content Area:(যে ডাটাগুলো show হবে তার container। .show class ব্যবহার করলে show/expanded থাকবে ।)

	The .accordion-collapse class is used for the collapsible content area.

	The .collapse class hides the content by default, and .show makes it visible (used for the first item in this example).

	The data-bs-parent ensures only one accordion item is open at a time (optional).

Accordion Body:(যেখানে ডাটা show হবে।)

	The .accordion-body class is used for the content inside the collapsible area.

কিভাবে accordion গুলো expanded রাখা যায় চাইলে।
<div id="collapseOne" class="accordion-collapse collapse show" data-bs-parent="#accordionExample"> 
এখান থেকে data-bs-parent উঠায় দিব।
<div id="collapseOne" class="accordion-collapse collapse show" data-bs-parent="#accordionExample"> 
where multiple items can stay open simultaneously), you need to remove the data-bs-parent attribute from the .accordion-collapse elements.
